# NLP-Sentiment-Analysis-Go

This repository contains a sentiment analysis microservice implemented in Go. It demonstrates efficient text processing, model integration, and deployment strategies suitable for production NLP applications.

## Features

- Fast text preprocessing.
- RESTful API for sentiment prediction.
- Dockerized for easy deployment.

## Getting Started

To run the service:

```bash
make build
make run
```

## API Endpoints

- `POST /analyze`: Analyze sentiment of input text.
